TYPOGRAPHY

This page demonstrates different font families, sizes, and styles used in web design.

1. Serif Fonts

Serif fonts have decorative lines at the ends of letters. Example: Times New Roman, Georgia.

2. Sans-Serif Fonts

Sans-serif fonts look clean and modern. Example: Arial, Helvetica, Verdana.

3. Monospace Fonts

Monospace fonts are often used for coding text. Every character takes the same width.

4. Decorative Fonts

Decorative fonts add creativity and fun to headings or short phrases.
